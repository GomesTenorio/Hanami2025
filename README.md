# CSV/XLSX Analytics API

API para processamento de arquivos CSV/XLSX e geração de relatórios analíticos.

## Requisitos

- Python 3.10+
- pip ou poetry

## Instalação

```bash
git clone <repo>
cd project-root
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows
pip install -r requirements.txt
```
